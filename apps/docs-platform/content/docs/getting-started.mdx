# Getting Started

Welcome to the Volteryde Platform! This guide will help you set up your local development environment.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** >= 20.0.0
- **pnpm** >= 8.0.0
- **Java** 17+ (for Spring Boot services)
- **Maven** 3.8+
- **Docker** >= 24.0
- **Docker Compose** >= 2.0

## Quick Start

### 1. Install Dependencies

```bash
pnpm install
```

This will install all dependencies for all workspaces in the monorepo.

### 2. Configure Environment

```bash
# Copy the example environment file
cp .env.example .env

# Edit .env with your local settings
# For local development, the defaults should work
```

All environment variables are managed centrally through `@volteryde/config` package.

### 3. Start Infrastructure

Start PostgreSQL, Redis, and Temporal using Docker Compose:

```bash
docker-compose up -d
```

This starts:
- **PostgreSQL** on `localhost:5432`
- **Redis** on `localhost:6379`
- **Temporal Server** on `localhost:7233`
- **Temporal UI** on `http://localhost:8080`

### 4. Start Services

Open separate terminal windows for each service:

#### NestJS Service

```bash
cd services/volteryde-nest
pnpm dev
```

Available at: http://localhost:3000/api/v1

#### Spring Boot Service

```bash
cd services/volteryde-springboot
mvn spring-boot:run
```

Available at: http://localhost:8080/api/v1

#### Temporal Workers

```bash
cd workers/temporal-workers
pnpm dev
```

## Verify Setup

Check that all services are running:

```bash
# NestJS Health Check
curl http://localhost:3000/api/v1

# Spring Boot Health Check
curl http://localhost:8080/api/v1/actuator/health

# View Temporal UI
open http://localhost:8080

# View Documentation
open http://localhost:3002
```
